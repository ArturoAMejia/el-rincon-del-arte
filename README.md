# El Rincon del Arte

Prototipo web de un marketplace para la venta de arte en Managua, Nicaragua

## Ejecutar proyecto

Primero, instalar las dependencias

```bash
npm i
# or
yarn
# or
pnpm i
# or
bun i
```

## Ejecutar servidor de desarrollo

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
